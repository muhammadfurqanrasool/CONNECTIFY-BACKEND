# CONNECTIFY BACKEND SERVER❤️

This is a backend RESTFUL API to give life to connectify!
This includes all features like Authentication with Cookies + JWT, uses mongoose API to interact with MONGODB and make models and schemas.

Make Sure to have:
- MongoDB installed and upbeen running!
- After clone, type in cmd, npm install
- Create .env file and all variables, PORT, MONGO_URI, SECRET_KEY
